Installation d'Angular

Prérequis

  • NodeJS

Installation d'Angular

  1. Installer Angular CLI

    npm install -g @angular/cli

  2. Tester l'installation

    ng --version

Outils

  • Votre éditeur de code préférer!

Extention disponible pour Angular

  • Angular Schematics: exécution de commande via un menu contextuelle
  • Angular Snippets: aide avec de l'auto-completion
  • Angular Language Service: aide avec de l'auto-completion
