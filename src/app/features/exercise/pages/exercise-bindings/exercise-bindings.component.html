Utilise tout ce que tu as appris dans le chapitre des bindings pour créer un mini formulaire!

Objectif

Pratiquer l'utilisation de l'interpolation, de la liaison de propriété, de la liaison d'événement et de la liaison d'attribut pour réaliser un "Template-driven form".

Template-driven Forms

Modifie le fichier src/app/features/exercise/pages/exercise-bindings/exercise-bindings.component.ts et src/app/features/exercise/pages/exercise-bindings/exercise-bindings.component.html pour réaliser un mini formulaire.

Consignes:

  • Le formulaire devra contenir 2 inputs: nom et prenom, et un bouton submit.
  • Le nom et prenom devront être afficher sur la page (voir image résultat).
  • Le bouton submit devra être désactivé si le nom ou le prénom est vide.
  • Le bouton submit devra afficher un message dans la console avec le nom et le prénom.
  • Bonus: utilise boostrap ou du css pour mettre un peu de style dans ton formulaire.
    🔎Boostrap est déjà en place dans le projet, il ne te reste qu'à utiliser ses noms de class
